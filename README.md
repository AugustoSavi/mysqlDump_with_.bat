## :question: Como usar esse repo?
    alterar os dados do arquivo backup.bat
    
    [database]
    [host]
    [password]
    [port]
    [directory]
    
    e depois basta executar o .bat

## :information_source: Attention

    Dont has space between -p and password -p[password]
